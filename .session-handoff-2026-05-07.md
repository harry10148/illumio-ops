## 環境踩過的坑（保留供後續 session 參考）

- **PYTHONPATH 雙路徑**: 用 `PYTHONPATH=$(pwd):$(pwd)/venv/lib/python3.12/site-packages python3 -m pytest <file> -v`
- **`/usr/bin/git` 直走**: 跳過 rtk wrapper，避免 `git log --graph` 等命令輸出被吃掉
- **rg 不在 PATH**: 用 `grep -rEn` fallback
- **rtk grep 偶爾過濾 multi-line output**: 對 `git show --stat`、`pytest -v` 等可能造成輸出誤差，必要時用 `/usr/bin/git` 直走
- **Click 8.3.3 `mix_stderr=False` removed**: 新版 CliRunner 自動分離 stderr/stdout，舊測試需移除參數
- **`ctx.exit()` 拋 click.exceptions.Exit**: 不是 `SystemExit`；`except SystemExit:` 捕不到 → 在外層 try/except Exception 會被吞，typed code 被覆蓋
- **illumio-ops.py dispatcher**: 新增 click subcommand 必須同步加入 `_CLICK_SUBCOMMANDS` set，否則 fall through 到 argparse
- **MJML 不採用**: 3 個 templates ~3 KB，Node.js dependency 不值；手動 patches 同樣達 rubric 目標
